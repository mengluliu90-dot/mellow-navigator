# Written Communication Request

## Use

Use this template when requesting written communication, accessible information, or reasonable adjustments.

The Equality Act 2010 is the primary UK legal framework for reasonable adjustments. The UN Convention on the Rights of Persons with Disabilities may be referenced as supporting context where it is useful, especially for accessible communication, information access, equal access to public services, participation, and independent living principles.

Do not create a separate disability-rights framework for each request. Use the rights reference only when it strengthens the communication.

## Template

Dear [Name / Team],

Please reply in writing where possible.

Written information helps me understand, check, and respond accurately.

Please confirm the next step and expected timescale.

Kind regards,

[Name]
