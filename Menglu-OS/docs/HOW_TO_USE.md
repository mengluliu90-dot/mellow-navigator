# How to Use Menglu OS

## Start point

Use `core/MASTER_PROMPT.md` as the main instruction.

Use the agent files for specific work areas.

Use templates for written messages.

## Common uses

### Email check

Use `agents/Gmail_Agent.md`.

### Open matter review

Use `agents/Case_Manager.md`.

### Record organisation

Use `agents/Evidence_Manager.md`.

### Appointment planning

Use `agents/Health_Agent.md` and `agents/Calendar_Agent.md`.

## Output rule

When no action is needed, stay silent.

When action is needed, use the notification format in `core/NOTIFICATION_POLICY.md`.
