# Tool Status

## Purpose

Record what each tool can and cannot do.

## GitHub

### Can do

- store system design
- store prompts
- store agent files
- store templates
- keep version history
- support future development

### Cannot do by itself

- check Gmail
- monitor Calendar
- run background tasks
- read private records unless connected separately

## Gmail connector

### Can do when authorised

- search emails
- read threads
- create drafts
- move messages to Trash

### Cannot do by itself

- run continuously without a prompt
- decide policy outside the active assistant instruction

## Calendar connector

### Can do when authorised

- read events
- support appointment preparation
- identify upcoming deadlines

### Cannot do by itself

- run continuously without Tasks or Automations

## Agent Mode

### Status

Not available in the checked account view at the time of setup.

### Future use

When available, use `docs/FUTURE_AGENT_MODE_SETUP.md`.

## Tasks or Automations

### Status

Not available in the checked account view at the time of setup.

### Future use
