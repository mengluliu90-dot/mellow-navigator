# Privacy Boundary

## Repository status

This repository is public.

## Do not add

- private identifiers
- contact details
- official numbers
- medical records
- bank details
- passwords
- private emails
- confidential documents

## Safe to add

- operating rules
- agent instructions
- reusable templates
- non-sensitive workflows
- implementation notes

## Live data

Use authorised connectors for live data checks.

Do not copy private live data into this repository.

## Sending rule

Drafts may be prepared.

Sending requires explicit confirmation.
